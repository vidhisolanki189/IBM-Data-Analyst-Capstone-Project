# IBM Data Analyst Capstone Project

## Project Overview
This project analyzes developer survey data to identify trends in programming languages, databases, cloud platforms, and demographics.

## Tools Used
- Python
- Pandas
- IBM Cognos Analytics
- Power BI

## Key Insights
- Python and JavaScript are highly popular programming languages.
- PostgreSQL and MySQL are widely used databases.
- Cloud technologies such as AWS are growing rapidly.
